# String.lsp
```
qr-string-removeWhiteSpace
    - Remove white space before/after sentences
    - 문장 앞/뒤 화이트 스페이스 제거한다.

qr-string-divide
    - Divide sentences using reference characters.
    - 참조 문자를 사용해서 문장을 나눕니다.

qr-string-lowerCase
    - Converts all strings to lowercase letters.
    - 문자열을 모두 소문자로 변환합니다.

```
