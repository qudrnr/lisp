# String.lsp
```
qr:RemoveWhiteSpace
    - 문장 앞/뒤 화이트 스페이스 제거한다.
    - Remove white space before/after sentences.

qr:RemoveWhiteSpace-front
    - 문장 앞부분 화이트 스페이스 제거한다.
    - Remove white space at the beginning of the sentence.

qr:RemoveWhiteSpace-rear
    - 문장 뒷부분 화이트 스페이스 제거한다.
    - Remove white space at the back of the sentence.

qr:FindPositionFirstNumber
    - 문장에서 첫번째 숫자의 위치를 찾는다.
    - Find the position of the first number in the sentence.

qr:DivideSentence
    - 참조 문자를 사용해서 문장을 나눕니다.
    - Divide sentences using reference characters.

qr:GetFirstWordfromString
    - 문장에서 첫 번째 단어를 얻습니다.
    - Get the first word from the sentence.

qr:GetLastWordfromString
    - 그 문장에서 마지막 단어를 얻습니다.
    - Get the last word from the sentence.

```
